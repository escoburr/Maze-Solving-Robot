Project: Ryerson University Project

Objective: http://www.ee.ryerson.ca/~lkirisch/ele758/labs/SimpleVideoGame[11-11-11].pdf
Video Game for VGA

File information:
DACclk -> Diving the clock to be in synchronous with the VGA monitor.
Hysnc -> Horizontal Sync of pixels on screen
Vsync -> Vertical Sync of pixels on screen
VidControl -> Main file to handle output from FPGA to the VGA monitor
ball -> Controls of the ball bounce
paddle -> How the paddles are controlled by switches
portmap -> VHDL function to implement entire design in one block
red8 -> Original clock for calculations
